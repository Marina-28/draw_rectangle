# :black_square_button: Draw_rectangle :black_square_button:
### The one of first School21's projects. The program prints a rectangle following certain rules.

### *Rules:*
* Top corners - ***A***
* Bottom corners - ***C***
* Sides - ***B***

### *Launch:*
```
./a.out x y 
```
*Comment: x and y some int numbers*


### *Example:*
```
%>./a.out 5 5 
ABBBA
B   B
B   B
B   B
CBBBC
```
